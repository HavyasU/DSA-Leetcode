# 709. To Lower Case

**Difficulty:** Easy

## Problem

Implement function `toLowerCase()` that converts a string to lowercase. All uppercase letters in the string should be converted to their corresponding lowercase letters; other characters should remain unchanged.

## Examples

**Example 1:**

```
Input: s = "Hello"
Output: "hello"
```

**Example 2:**

```
Input: s = "here"
Output: "here"
```

## Constraints

- `1 <= s.length <= 10^5`
- `s` consists of ASCII characters.
